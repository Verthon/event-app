1. creating new event, information required

  * event description
  * localization
  * date(date-picker)
  * photo (no upload, only links)
  * category
2. displaying all events
3. search function
4. sorting events
5. deleting event
6. editing created event
7. saving app state to database
8. Clean UI and animations
