## Technologies and
- Ionic 4
- React.js
- Redux Toolkit
- Typescript
- Styled Components
- Firestore


## Setup

1. clone repository `git clone https://github.com/Verthon/event-app.git`
2. install dependencies `npm install`
3. run react dev server `npm start`
